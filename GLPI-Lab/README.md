
# 🖥️ GLPI Lab – Gestion du Support Informatique

## 🎯 Objectif

Mettre en place une plateforme GLPI pour la gestion des tickets et l’inventaire du parc informatique.

---

## 🏢 Utilité en entreprise

GLPI permet de :

- gérer les incidents informatiques
- suivre les demandes utilisateurs
- inventorier les équipements
- améliorer le support IT

---

## 🛠️ Technologies utilisées

- Linux
- Apache
- MySQL
- PHP
- GLPI

---

## ⚙️ Étapes d'installation
1. Installation Apache
2. Installation MySQL
3. Installation GLPI
4. Configuration de la base de données
5. Création des utilisateurs
